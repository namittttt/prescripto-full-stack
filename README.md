# 🏥 Doctor Appointment Management System

A full-stack web application that allows patients to book appointments and doctors to manage schedules efficiently.

## 🚀 Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## ✨ Features
- User authentication (JWT)
- Book & manage appointments
- Doctor dashboard
- Admin panel
- Payment integration

## 📂 Project Structure
- frontend/
- backend/
- admin/

## ⚙️ Setup
```bash
cd backend && npm install
cd frontend && npm install
